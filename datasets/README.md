# Datasets 使用说明

本目录仅用于存放本地数据集压缩包与解压内容，默认不会上传到 GitHub。

## 请下载并放置以下压缩包到本目录（`datasets/`）

- `DRIVE.zip`
  - 下载地址：https://www.isi.uu.nl/Research/Databases/DRIVE/
- `kvasir-seg.zip`
  - 下载地址：https://datasets.simula.no/kvasir-seg/
- `Task04_Hippocampus.tar`
  - 下载地址：https://github.com/Project-MONAI/MONAI-extra-test-data/releases
- `Task09_Spleen.tar`
  - 下载地址：https://github.com/Project-MONAI/MONAI-extra-test-data/releases

## 文件命名规范（必须一致）

请确保文件名与以下名称完全一致：

- `DRIVE.zip`
- `kvasir-seg.zip`
- `Task04_Hippocampus.tar`
- `Task09_Spleen.tar`

脚本会按以上固定文件名查找并解压。
