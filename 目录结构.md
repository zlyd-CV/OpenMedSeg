【测试数据集策略（首次自动下载 + 工业化缓存）】

- 缓存根目录：
  - Windows: %LOCALAPPDATA%/MedAI-ModelZoo/datasets/
  - 其他系统: ~/MedAI-ModelZoo/datasets/

- 2D 网络测试数据：
  - 简单难度: DRIVE（视网膜血管分割）
    -> 缓存目录: datasets/task2_2d_simple/DRIVE/
  - 中等难度: Kvasir-SEG（胃肠道息肉分割）
    -> 缓存目录: datasets/task3_2d_medium/Kvasir-SEG/

- 3D 网络测试数据：
  - 简单难度: MSD Task04_Hippocampus
    -> 缓存目录: datasets/task2_3d_simple/Task04_Hippocampus/
  - 中等难度: MSD Task09_Spleen
    -> 缓存目录: datasets/task3_3d_medium/Task09_Spleen/
